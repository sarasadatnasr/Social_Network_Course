# Social Network HW1
---
This repository contains the implementation and analysis of various network models and phenomena explored in the Social Network course. It includes both theoretical and practical exercises related to network construction, scaling analysis, and the study of random and generative graph models.

## Contents

1. **Theoretical Questions**

   * **Watts-Strogatz Model**: Network construction and simulation, clustering coefficient for rewired networks.
   * **Snobbish Network**: Analysis of global connectivity and the scaling of shortest path lengths in networks with different intra- and inter-community link probabilities.

2. **Implementation Questions**

   * **Small-World Phenomena**: Simulation of random networks and the analysis of scaling behavior in different lattice configurations (1D, 2D, 3D).
   * **Generative Graph Models**: Deterministic construction of scale-free networks and the exploration of the Random Genetic (RG) model.
   * **Network Evolution and Phase Transitions**: Simulation of Erdős-Rényi network growth, critical threshold identification, and finite-size effects.

## How to Run

1. Clone this repository:
   `git clone https://github.com/yourusername/Social-Network-HW1.git`

2. Install the required dependencies:
   Ensure you have Python 3.x installed, then run:
   `pip install -r requirements.txt`

3. Run the simulations and analyses in the provided Jupyter notebooks or Python scripts.

## Results

The repo includes simulation outputs and figures detailing the scaling of average path lengths, network robustness under node removal, and the emergence of the giant component in evolving networks.
