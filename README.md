# Social Network HW02

This repository contains the implementation and analysis for Homework 2 of the Social Network course. It covers both theoretical and practical exercises related to centrality measures, network analysis, and ranking algorithms.

## Contents

### Theoretical Questions

* **Structural Index**: Explanation of functions and determination of structural indices.
* **Distances and Neighbors**: Network analysis for optimal node placement in various scenarios.
* **Stress Centrality**: Analysis of node and edge stress centrality.

### Implementation Questions

* **Political Power Analysis**: Structural analysis of a global political network, including centrality calculations and gap analysis.
* **Ranking Algorithms**: Comparative analysis of HITS and PageRank algorithms on the Wiki-Vote dataset.

## How to Run

1. Clone this repository:

   ```
   git clone https://github.com/yourusername/Social-Network-HW02.git
   ```

2. Install the required dependencies:

   ```
   pip install -r requirements.txt
   ```

3. Run the Python scripts in the **Q1/** and **Q2/** directories for the corresponding analyses.

## Results

The repository includes outputs for centrality measures, gap analysis plots, and comparisons between HITS and PageRank rankings.
