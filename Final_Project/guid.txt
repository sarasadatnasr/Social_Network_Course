1. telegram_10_to_24_dey_edge.csv

This file contains data about edges in the network, with the following columns:

node1: The source node of the edge (ID of the referring account).

node2: The target node of the edge (ID of the original publisher).

directed: Always TRUE indicating a directed edge.

weight: The weight of the edge, which indicates how many times the content was referenced or shared.

2. telegram_10_to_24_dey_node.csv

This file contains information about the nodes, including:

nodedef_name: Unique account or channel identifier (numeric).

label: The username of the account/channel (could be dynamic due to changes in usernames over time).

views: Total views of the posts made by the account/channel.

followers: The number of followers of the account/channel at the moment of data collection.

importance: A combined non-linear number representing the importance of the account based on posts, views, and interactions.

posts: The number of posts/tweets made by the account/channel.

positive: The number of positive posts (expressing approval or joy).

negative: The number of negative posts (expressing dissatisfaction or anger).

neutral: The number of neutral posts (no strong opinion).

3. telegram_15_dey.csv, telegram_16_dey.csv, ..., telegram_20_dey.csv

These files contain the actual posts made on Telegram. The columns include:

platform: The platform (Telegram in this case).

name: The publisher's name.

username: The account's username.

follower: The total number of followers of the account at the time of the post.

link: The link to the post.

date: The date of the post (Gregorian calendar).

date(jalali): The date in Jalali (Persian) calendar.

text: The content of the post.

type: The type of post (e.g., original, reshared, quoted, reply).

copy: The number of copies of this post without reference.

impression: The number of views of the post.

engagement: The total user interactions (likes, shares, comments).

comment: The number of comments on the post.

like: The number of likes on the post.

forward: The number of times the post was forwarded.

sentiment: The sentiment of the post (positive, negative, neutral).

avatar: The link to the user's avatar.
